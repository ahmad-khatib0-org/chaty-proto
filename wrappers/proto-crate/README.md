# Chaty Proto - Rust Crate

Generated gRPC protobuf definitions for Chaty services.

## Usage

Add to your `Cargo.toml`:

```toml
[dependencies]
chaty-proto = "0.1.0"
tonic = "0.13"
prost = "0.13"
```

## Features

- Generated with Tonic 0.13
- Includes message definitions and gRPC service stubs
- Server support with Tonic
- Full serde support for serialization
